# Dual-Cell-Ion-Battery-Controler

## Features:

• Dual-Cell Ion Battery Controler 18650

• MCU S-82B2AAA-I8T1U7
   
• over charge protection

• over discharge protection

• over current protection

## PCB SPECS:

•	Size: 92.3 X 55.2 MM

•	Layer Count: 1

•	Thickness: 1.6MM.
